# lone-ranger

Установи свежую nodejs

Клонируй репозиторий 
```bash
git clone https://github.com/ildarka/lone-ranger.git
```

Установи глобальные модули
```bash
npm install -g bower
npm install -g gulp
```

Установи зависимости
```bash
npm install
```

Запусти сервер
```bash
gulp
```
Если все прошло успешно — морда будет доступна по адресу
http://localhost:3000
